
# Real-Time Audio Transcription and Sentiment Analysis

This application transcribes audio input and analyzes the sentiment of the transcribed text using state-of-the-art deep learning models.

## Features
- Transcribe speech to text using OpenAI's Whisper model.
- Perform sentiment analysis using Cardiff NLP's Twitter RoBERTa model.
- Supports audio input via microphone or file upload.
- Provides real-time transcription and sentiment analysis results.

## Installation & Usage
### Running Locally
1. Clone this repository:
   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Open the local URL provided by Gradio to use the app.

### Running on Hugging Face Spaces
1. Create a new Space on Hugging Face and choose the `Gradio` template.
2. Upload the files (`app.py`, `requirements.txt`, and `README.md`).
3. The app will automatically build and deploy.

## Requirements
- Python 3.8+
- Torch (PyTorch)
- Transformers
- Gradio
